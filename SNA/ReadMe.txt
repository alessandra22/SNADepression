Suggested operations for correctly execute the program:

> create a project in PyCharm
> copy SNA as a directory inside the project
> be sure to have installed all the imports in the files:
    - numpy
    - pandas
    - networkx
    - matplotlib
    - scipy
    - sklearn
> right click on project.py and press "run 'project'"
> output will appear on the Run tab below
> plots will appear on the SciView tab on the right
